# 🗺️ Spatial Indexing System Using QuadTree

A C++ implementation of a Spatial Indexing System using the 
QuadTree data structure for efficient location-based queries.


- **Yusra Rana** — 
  

## 🏫 Project Info
- **University:** Fatima Jinnah University, Rawalpindi
- **Course:** Data Structures (DS)
- **Semester:** 3rd Semester

## 📋 Features
- 📍 **Insert Location** — Add cabs, restaurants, users, hospitals
- 🔍 **Search by Name/ID** — Find any location instantly
- 📊 **Range Query** — Count locations in any area
- 🚖 **Find Nearest Cab** — Ride-sharing simulation
- 🍕 **K Nearest Restaurants** — Food delivery simulation
- 👥 **Nearby Friends** — Social media simulation
- 🏥 **Nearest Hospital** — Emergency response simulation
- 🌳 **Tree Visualization** — View QuadTree structure

## 🛠️ Built With
- Language: **C++**
- Concepts: **OOP, Recursion, Spatial Indexing**
- Tool: **Dev C++ / VS Code**

## ⏱️ Time Complexity
| Operation | Average | Worst |
|-----------|---------|-------|
| Insert | O(log n) | O(n) |
| Search | O(log n) | O(n) |
| Range Query | O(log n + k) | O(n) |
| Nearest Neighbor | O(log n) | O(n) |
